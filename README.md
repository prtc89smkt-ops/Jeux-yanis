# ✈️ Yanis Chris

Bienvenue dans **Yanis Chris**, un jeu d'avion en 2D développé en HTML, CSS et JavaScript.

## 🎮 Description

Pilote ton avion, esquive les ennemis et détruis-les grâce au tir automatique. Le jeu est jouable sur ordinateur (souris) et sur mobile (écran tactile).

## ✨ Fonctionnalités

- ✈️ Avion contrôlable à la souris ou au toucher
- 🔫 Tir automatique
- 👾 Apparition d'ennemis
- 💥 Détection des collisions
- 🏆 Système de score
- ❤️ Système de vies
- 📱 Compatible mobile et ordinateur
- 🌌 Interface simple et moderne

## 🚀 Installation

1. Télécharge ou clone le projet :

```bash
git clone https://github.com/VOTRE-NOM/Yanis-Chris.git
```

2. Ouvre le dossier du projet.

3. Lance le fichier `index.html` dans ton navigateur.

Aucune installation supplémentaire n'est nécessaire.

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript (Canvas API)

## 🎯 Contrôles

### Ordinateur

- 🖱️ Déplace la souris pour piloter l'avion.
- 🔫 Les tirs sont automatiques.

### Mobile

- 👆 Fais glisser ton doigt sur l'écran pour déplacer l'avion.
- 🔫 Les tirs sont automatiques.
